# D'Oh! Memory 🍩

A Simpsons-themed memory card matching game built with React.

**Play:** [A Simpsons Memory Game](https://asimpsonsmemorygame.netlify.app)
## [Live Demo](https://youtu.be/hjVpLDoU9sA)


## About

D'Oh! Memory is a classic memory card matching game reimagined with characters from The Simpsons. Flip cards, find matching pairs, and try to beat your best score, all while avoiding clicking the same card twice in a row .

## Features

- Simpsons-themed card designs featuring iconic characters
- Score tracking to challenge yourself on replay
- Responsive layout for both desktop and mobile play
- Smooth, interactive card-flip animations
- Built with reusable React components and clean state management

## Tech Stack

- **React** – component-based UI and game state management
- **JavaScript (ES6+)**
- **CSS** – styling and animations
- **Netlify** – deployment and hosting

## Deployment

This project is deployed on Netlify, with the build command `npm run build` and publish directory `dist`.
